Этот проект выполняет автоматизированное тестирование функциональности регистрации на сайте Stellar Burgers, 
используя Selenium и pytest.

Тесты проверяют:
1) регистрацию
2) логин с разных экранов входа
3) навигацию по конструктору бургеров
4) переход в личный кабинет
5) логаут

Для запуска тестов на локальной машине необходимы:

Python (версии 3.7 или выше)
Selenium
WebDriver для используемого браузера (например, ChromeDriver для Google Chrome)

Установка зависимостей:
pip install selenium pytest
